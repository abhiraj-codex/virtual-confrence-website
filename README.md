# Virtual Conference Website

A modern and responsive **Virtual Conference Landing Page** built using HTML, CSS, and JavaScript.  
This project showcases a clean UI for hosting and promoting online events or conferences.

The website includes a responsive navigation bar, hero section, sponsor logos, smooth animations, and a mobile-friendly menu.

## Features

- Responsive navigation bar
- Mobile hamburger menu
- Smooth animations using AOS
- Modern hero section
- Sponsor/company logo section
- Sticky header with blur effect on scroll
- Clean and modern UI design
